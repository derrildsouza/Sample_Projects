# System Requirements Specification (SYS.1)

## 1. Overview
High-level system requirements defining the business needs and operational environment.

## 2. Functional System Requirements

| ID | Description |
| :--- | :--- |
| **SYS_REQ_001** | The system shall provide a client-server mechanism for time synchronization. |
| **SYS_REQ_002** | The system shall support IPv4 network communication. |
| **SYS_REQ_003** | The system shall allow execution on Linux-based operating systems. |
| **SYS_REQ_004** | The system shall effectively handle command-line inputs for configuration. |

## 3. Non-Functional System Requirements

| ID | Description |
| :--- | :--- |
| **SYS_NFR_001** | The system shall respond to client requests within 1 second under normal load. |
| **SYS_NFR_002** | The system code shall be open-source (MIT License). |
